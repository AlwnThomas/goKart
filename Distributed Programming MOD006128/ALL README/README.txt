Part 1 - Kart Spinning Animation

50x50 pixels go-kart wviewed from above, 16 versions of the image
rotated at the centre. Each frame updates to rotate the image
22.5 degrees. Two kart images given, one green and another one red.

How to run?

1. Place the 16 kart images (each image rotated 22.5 degrees
   more than the next one) inside goKart/resources/Kart1/...

2. Run code KartSpinning.java (make sure to also have 
   KartAnimationPanel.java) in the same source folder.

3. You should have a kart spinning at its center!
